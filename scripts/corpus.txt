# Training corpus for STE fine-tuning of binary GPT-2.
# Diverse, coherent English sentences (factual + narrative), public-domain
# / common-knowledge. One sentence per line. Aim: 8-32 BPE tokens each so the
# tokenizer keeps them (see scripts/build_corpus.c).
# Topics mixed: geography, science, history, daily life, narrative, technology,
# nature, proverbs — to give the model varied contexts like GPT-2's pretraining.
